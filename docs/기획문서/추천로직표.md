# 가계부 기반 저축 운영 추천 서비스 MVP 추천 로직 표

## 1. 문서 개요

본 문서는 MVP 단계에서 사용자의 입력값을 바탕으로 어떤 저축 운영 방식을 추천할지 정의하기 위한 문서이다.  
추천 로직은 사용자에게 종목이나 개별 상품을 제안하는 것이 아니라, 현금흐름과 목표에 적합한 자금 운영 방식을 안내하는 것을 목적으로 한다.

본 문서의 범위는 다음과 같다.

- MVP 추천 판단 기준 정의
- 추천 조합 분기 기준 정의
- 추천 결과의 기본 형식 정의
- 후속 고도화 시 확장 가능한 기준점 정리

## 2. MVP 추천 로직 기본 원칙

- 추천은 종목이 아닌 운영 방식 단위로 제공한다.
- 추천은 저축 성향만이 아니라 현금흐름 안정성, 비상금 보유 수준, 목표 성격을 함께 고려한다.
- 비상금이 부족하거나 잉여자금이 낮은 경우에는 투자형 요소보다 유동성 확보를 우선한다.
- 사용자의 입력값만으로 해석이 어려운 경우에는 보수적 추천을 기본값으로 한다.
- 추천 결과는 반드시 추천 이유와 함께 제공한다.

## 3. MVP 입력값 및 판단 변수

### 3.1 MVP 필수 입력값

- 월 실수령액
- 월급일
- 월 고정비
- 월 변동지출
- 현재 비상금 보유액
- 저축 목표
- 저축 성향

### 3.2 핵심 판단 변수

| 변수명 | 정의 | 산식 또는 판단 기준 |
|---|---|---|
| 월 잉여자금 | 매월 남는 금액 | 월 실수령액 - 월 고정비 - 월 변동지출 |
| 고정비 비중 | 소득 대비 반복 지출 비중 | 월 고정비 / 월 실수령액 |
| 생활비 기준액 | 월 생계 유지에 필요한 기준 금액 | 월 고정비 + 월 변동지출 |
| 비상금 목표액 | 우선 확보가 필요한 비상 자금 기준 | 생활비 기준액 x 3개월 |
| 비상금 충족률 | 비상금 확보 수준 | 현재 비상금 보유액 / 비상금 목표액 |
| 목표 성격 | 자금 운용의 목적 | 비상금, 생활 안정, 단기 목표, 중장기 준비 등 |

## 4. 저축 여력 단계 정의

| 단계 | 조건 | 해석 |
|---|---|---|
| 부족 | 월 잉여자금 0원 이하 | 저축 확대보다 지출 조정이 우선 |
| 낮음 | 월 잉여자금이 양수이나 생활비 대비 여유가 작음 | 보수적 저축 구조 필요 |
| 보통 | 월 잉여자금이 안정적으로 확보됨 | 저축과 일부 분산 운용 병행 가능 |
| 높음 | 월 잉여자금이 충분하고 고정비 압박이 낮음 | 목표 중심 자금 배분 확대 가능 |

### 4.1 MVP 단계 수치 기준

MVP에서는 아래 기준으로 단순 분류한다.

- 부족: 월 잉여자금 0원 이하
- 낮음: 월 잉여자금 1원 이상, 월 실수령액의 10% 미만
- 보통: 월 잉여자금이 월 실수령액의 10% 이상 25% 미만
- 높음: 월 잉여자금이 월 실수령액의 25% 이상

## 5. 비상금 상태 정의

| 단계 | 조건 | 해석 |
|---|---|---|
| 없음 | 비상금 충족률 0% | 즉시 유동성 확보 필요 |
| 부족 | 비상금 충족률 1% 이상 100% 미만 | 비상금 우선 확보 필요 |
| 충족 | 비상금 충족률 100% 이상 | 중기적 저축 및 분산 운용 검토 가능 |

## 6. 저축 성향 정의

| 성향 | 정의 | 기본 방향 |
|---|---|---|
| 안정형 | 원금 변동 가능성을 최소화하려는 성향 | 파킹통장, 예적금 중심 |
| 균형형 | 안정성과 성장 가능성을 함께 고려하는 성향 | 예적금과 적립식 운용 병행 |
| 적극형 | 장기적 자금 운용 확대에 비교적 수용적인 성향 | 적립식 운용 비중 확대 가능 |

## 7. 추천 방식 분류

MVP에서는 추천 결과를 아래 운영 방식 중 하나 또는 조합으로 제시한다.

- 유동성 확보형: 파킹통장 중심 운영
- 안정 저축형: 파킹통장 + 적금 조합
- 균형 저축형: 적금 + ISA 적립 조합
- 분산 운용형: 적금 + ISA 적립 + 일반 적립식 운용 조합
- 지출 조정 우선형: 신규 저축 확대보다 소비 구조 점검 우선

## 8. 기본 추천 로직 표

| 저축 여력 | 비상금 상태 | 저축 성향 | 추천 방식 | 기본 추천 구조 | 추천 이유 |
|---|---|---|---|---|---|
| 부족 | 없음/부족/충족 | 전체 | 지출 조정 우선형 | 신규 저축 확대 보류 | 잉여자금이 없어 우선적으로 지출 구조 조정이 필요함 |
| 낮음 | 없음 | 전체 | 유동성 확보형 | 파킹통장 중심 | 비상금이 전혀 없어 즉시 사용 가능한 자금 확보가 우선임 |
| 낮음 | 부족 | 안정형 | 안정 저축형 | 파킹통장 비중 높음 + 소액 적금 | 잉여자금이 적고 비상금이 부족하여 안정적 확보가 우선임 |
| 낮음 | 부족 | 균형형/적극형 | 안정 저축형 | 파킹통장 중심 + 소액 적금 | 성향보다 비상금 확보가 우선이며 투자형 확대는 이른 단계임 |
| 낮음 | 충족 | 안정형 | 안정 저축형 | 적금 중심 + 소액 파킹통장 | 잉여자금은 작지만 비상금이 확보되어 안정적 저축이 적합함 |
| 낮음 | 충족 | 균형형/적극형 | 균형 저축형 | 적금 중심 + 소액 ISA 적립 | 제한된 여력 내에서 소규모 분산 운용이 가능함 |
| 보통 | 없음 | 전체 | 유동성 확보형 | 파킹통장 우선 + 적금 보조 | 비상금 미보유 상태에서는 유동성 우선 확보가 필요함 |
| 보통 | 부족 | 안정형 | 안정 저축형 | 파킹통장 + 적금 | 비상금 보완과 안정적 저축을 병행하는 구조가 적절함 |
| 보통 | 부족 | 균형형 | 균형 저축형 | 파킹통장 + 적금 + 소액 ISA 적립 | 비상금 보완을 유지하며 점진적 분산 운용 가능 |
| 보통 | 부족 | 적극형 | 균형 저축형 | 파킹통장 + 적금 + 소액 ISA 적립 | 적극 성향이어도 비상금 부족 단계에서는 보수적 병행 구조가 적절함 |
| 보통 | 충족 | 안정형 | 안정 저축형 | 적금 중심 + 파킹통장 보조 | 비상금이 확보되었으므로 안정적 누적 저축이 적합함 |
| 보통 | 충족 | 균형형 | 균형 저축형 | 적금 + ISA 적립 | 안정성과 분산 운용을 함께 반영한 구조가 적절함 |
| 보통 | 충족 | 적극형 | 분산 운용형 | 적금 + ISA 적립 + 일반 적립식 운용 | 잉여자금과 비상금이 확보되어 적립식 분산 운용 확대 가능 |
| 높음 | 없음 | 전체 | 유동성 확보형 | 파킹통장 우선 | 여력이 있어도 비상금 미보유 상태에서는 유동성 확보가 선행되어야 함 |
| 높음 | 부족 | 안정형 | 안정 저축형 | 파킹통장 + 적금 | 여유자금을 활용한 빠른 비상금 보완이 적절함 |
| 높음 | 부족 | 균형형/적극형 | 균형 저축형 | 파킹통장 + 적금 + ISA 적립 | 비상금 보완과 중장기 운용을 병행할 수 있음 |
| 높음 | 충족 | 안정형 | 안정 저축형 | 적금 중심 + 파킹통장 | 장기적 안정 저축 확대에 적합한 상태임 |
| 높음 | 충족 | 균형형 | 분산 운용형 | 적금 + ISA 적립 + 일반 적립식 운용 | 안정성과 성장성을 동시에 고려할 수 있음 |
| 높음 | 충족 | 적극형 | 분산 운용형 | ISA 적립 + 일반 적립식 운용 + 일부 안정 자금 유지 | 중장기 자금 운용 확대가 가능한 상태임 |

## 9. 목표 기반 보정 로직

기본 추천은 잉여자금, 비상금, 성향 기준으로 결정하되, 저축 목표에 따라 결과를 보정한다.

| 목표 | 우선 고려 사항 | 보정 원칙 |
|---|---|---|
| 비상금 확보 | 유동성, 즉시 사용 가능성 | 파킹통장 비중 확대 |
| 생활 안정 | 변동성 최소화 | 적금 비중 확대 |
| 여행/소비 예정 | 단기 자금 확보 | 파킹통장 또는 단기 적금 중심 |
| 주거/결혼 준비 | 중기 자금 누적 | 적금 + ISA 적립 병행 가능 |
| 투자 준비 | 장기 운용 가능성 | 비상금 충족 이후 적립식 운용 비중 확대 |

## 10. 권장 저축액 산출 원칙

MVP에서는 복잡한 알고리즘 대신 이해 가능한 단순 규칙을 사용한다.

| 항목 | 산출 원칙 |
|---|---|
| 안전 저축액 | 월 잉여자금의 50% |
| 권장 저축액 | 월 잉여자금의 70% |
| 도전 저축액 | 월 잉여자금의 90% |

단, 월 잉여자금이 0원 이하일 경우 세 항목 모두 0원으로 계산한다.

## 11. 자금 배분 비율 가이드

### 11.1 유동성 확보형

- 파킹통장 70~100%
- 적금 0~30%
- ISA 및 적립식 운용 0%

### 11.2 안정 저축형

- 파킹통장 20~50%
- 적금 50~80%
- ISA 및 적립식 운용 0~20%

### 11.3 균형 저축형

- 파킹통장 10~30%
- 적금 40~60%
- ISA 적립 20~40%

### 11.4 분산 운용형

- 파킹통장 10~20%
- 적금 20~40%
- ISA 적립 20~40%
- 일반 적립식 운용 20~40%

## 12. MVP 이후 확장 포인트

- 목표 기간, 투자 경험, 지출 변동성을 반영한 세부 보정 로직 추가
- 부채 유무에 따른 우선순위 재조정
- 사용자 유형별 추천 문구 세분화
- 월별 실제 기록 데이터를 반영한 추천 자동 갱신

## 13. 결론

MVP 추천 로직은 복잡한 금융 조언이 아니라, 사용자의 월별 현금흐름과 비상금 수준을 기준으로 저축 운영 우선순위를 정리해주는 데 목적이 있다.  
따라서 초기 단계에서는 예외 상황을 과도하게 세분화하기보다, 이해 가능하고 일관된 규칙을 통해 보수적이고 신뢰 가능한 추천 경험을 제공하는 것이 중요하다.
